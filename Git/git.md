語法
======

#### 'git init'
創建一個git追蹤，並可開始使用status, add....<br>
#### git status
查看是否有新版本<br>
#### git add
使新版本加入追蹤。<br>
+ git add filename：使filename加入追蹤<br>
+ git add .：使全部檔案的新版本加入追蹤<br>
#### git commit -m "title" -m "description"
將暫存區的內容內容提交至git版本控制庫中<br>
